2. 最初的作法是將 (x, y) 座標旋轉至 (a, b) 座標，點不旋轉，但最後參考座標還是在 (x, y)
3. 現在是將 (x, y) 與 (a, b) 座標重疊，旋轉點，最後參考座標會變到 (a, b)
4. 兩個方法間 theta 差了一個負號，也是造成錯誤的原因
